# Millerapps Tap

## How do I install these formulae?

`brew install millerapps/tap/<formula>`

Or `brew tap millerapps/tap` and then `brew install <formula>`.

Or, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "millerapps/tap"
brew "<formula>"
```
